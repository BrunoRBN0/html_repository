ao criar um arquivo em html, necessário ele ser index.html
<h1>-><h6> tamanho do titulo(não pode haver dois do mesmo na mesma estrutura)
<hr/> separar titulos
<p> paragrafo
<a href="link"> para adicionar um link
<b><strong> = é para deixar em negrito, em um só vizual, no outro é semantica
<br> = quebra de linha
<i><em> italico, em um só vizual, no outro é semantica
<mark> marca texto
<del> = da um corte na palavra como se tivesse deletada
<small> = diminuir a fonte palavra
<span> estilização em css, ou enjetado via js
<blockquote> aplicação de margem
<header> separar como cabeçalho
<nav> seção da pagina
<footer> roda pé
&copy; = simbolo de copyright
<main> conteudo principal
<article> artigos das sections
<section> separador de seções
<aside> bara lateral, costuma ser utilizado com css
<div> para fazer modificações em partes especificas do códigos, podendo se usado com css ou js

Lista:
<ol> lista ordenada
<li> para adicionar na lista algo
<ul> lista não ordenada

Tabela:
<table> estrutura de tabela
<thead> cabeçalho da tabela
<tr> linha da tabela(Ele vai indo para baixo)
<th> cabeçalho da tabela
<tbody> estrutura do corpo da tabela
<td>coluna onde colocar a linha (começa no primeiro, e conforme tu for colocando em mais linhas, vai indo para o lado)
<tfoot> rodapé da tabela
<caption> Descreve doq se trata a tabela

Formulário:
<form> Criar estrutura de Formulário
<input type=""placeholder="oq vai aparecer na linha ex: Seu nome"/> escolher o tipo de campo do formulário que vai ser adicionado
<label> para inforar oq o campo necessita(ele aparece antes da linha informando oq se trata).
<label for="id do campo"> o label tem a possibiliadade de colocar o id, para só de clicar encima do nome dele, ele identifique que tu está se tratando da linha de email(necessário colocar também no input que se relacione ao id que foi posto no label)
id = vai dentro de ipunt, para localizr o "for" posto no label
max - min = delimita o limite no type "number" par quanto começar e quanto ir
step = faz ele ir pulando de tanto em tanto(ex: step="2" 2-4-6-8-10)
type="radio" = ele dá opções para marcar
name="algo ai" = para deixar opção unica a pergunta
<select> selecionar um item das opções
<select multiple> selecionar mais de um item das opções
<option> quais as resposta para selecionar
<textarea> caixa de texto
<button type=""> ele coloca um botão, e maneira que tu selecionar o tipo do botão, ele reage de uma forma
